# Inloggen

Zo krijg je toegang tot het Fruitvriendjes Admin Panel.

## Toegangsrechten

Om in te loggen op het admin panel heb je een email-adres nodig van een van deze domeinen:

- **handihow.com**
- **eduprompt.nl**
- **schoolfruit.nl**

Anonieme gebruikers en gebruikers van andere domeinen worden automatisch geweigerd.

## Inlogmethoden

Je kunt inloggen op twee manieren:

### 1. Met Google Account

- Klik op "Sign in with Google"
- Kies je werkaccount
- Je wordt automatisch ingelogd als je emaildomein is toegestaan

### 2. Met Email en Wachtwoord

- Voer je email-adres en wachtwoord in
- Klik op "Sign in"

:::tip
Je kunt je wachtwoord resetten via de "Forgot password" link als je deze bent vergeten.
:::

## Admin Panel URL

Het admin panel is beschikbaar op: **https://fruitvriendjes-35c8c.web.app/admin**

## Problemen met inloggen?

Als je problemen hebt met inloggen:

1. **Controleer je emaildomein** - Alleen gebruikers van toegestane domeinen kunnen inloggen
2. **Wis je browser cache** - Soms helpt het om je cache te legen
3. **Probeer een andere browser** - Test in een andere browser
4. **Neem contact op** - Als niets werkt, neem contact op met het ontwikkelteam

:::warning Belangrijk
Anonieme accounts worden niet toegestaan in het admin panel. Je moet inloggen met een bestaand account van een toegestaan domein.
:::
