## Data Bekijken

### Tabelweergave

Elke collectie toont een tabel met data:

- **Kolommen** - Verschillende velden per collectie
- **Sorteren** - Klik op kolomkop om te sorteren
- **Filters** - Gebruik filteropties om data te filteren

### Detail Weergave

Klik op een rij om de detailweergave te openen:

- Alle velden zichtbaar (ook verborgen velden)
- Volledig geformatteerde tekst
- Afbeeldingen in volle grootte
- Relaties naar andere records

### Zoeken

Gebruik de zoekfunctie:

1. Klik op het zoek icoon
2. Type je zoekterm
3. Druk op Enter
4. Bekijk gefilterde resultaten

### Regelmatig Controleren

- Check wekelijks nieuwe recepten
- Monitor chatbot kwaliteit
- Bekijk gebruikerstrends

### Privacy

- Gebruikers-IDs zijn verborgen in tabellen
- Deel geen persoonlijke data extern
- Bekijk alleen data waar je toegang tot nodig hebt

### Performance

- Gebruik filters voor grote datasets
- Sluit detail weergaves na gebruik
- Refresh pagina als tabel traag reageert

## Veelgestelde Vragen

### Kan ik data bewerken?

Nee, alle collecties zijn read-only. Je kunt data alleen bekijken.

### Hoe vaak wordt data ververst?

Data in het admin panel is real-time. Elke wijziging in de app is direct zichtbaar.

### Kan ik data verwijderen?

Nee, deleties zijn niet mogelijk via het admin panel. Neem contact op met ontwikkelaars indien nodig.

### Hoelang wordt data bewaard?

Alle data wordt permanent bewaard tenzij expliciet verwijderd door ontwikkelaars.
